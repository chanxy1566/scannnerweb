# 售后订单管理系统（Web版）

## 1. 项目简介
本系统用于售后订单的扫码录入、数据匹配、查询导出及统计看板。
包含扫码管理、订单管理、商品/套装档案、数据分析、用户管理、操作日志等模块。

## 2. 环境要求
- Python 3.8+
- Windows / Linux（Windows 下使用 waitress 部署，Linux 推荐 gunicorn）
- 扫码枪（免驱，模拟键盘输入）

## 3. 安装与配置

### 3.1 获取代码
将项目文件夹放置到目标路径，例如 `D:\after-sales`

### 3.2 安装依赖
```bash
pip install -r requirements.txt
